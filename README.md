UniShedr - Frequently Asked Questions (FAQ)

What is UniShedr? UniShedr is a multifunctional Discord bot designed to enhance your server experience with moderation tools, user engagement features, and monetization support.

How do I invite UniShedr to my server? You can invite UniShedr using the official invite link provided by the bot developer. Make sure you have the necessary permissions to add bots to your server.

What permissions does UniShedr need? UniShedr requires permissions such as:

Manage Messages

Kick Members

Ban Members

Read Message History

Send Messages

Embed Links

Use Slash Commands These permissions allow the bot to perform moderation and interactive commands effectively.

How does the monetization system work? UniShedr supports server monetization by restricting access to certain features based on server size and Patreon subscription status. Server owners must link their Patreon account and manage their subscription tiers to unlock premium features.

How can I link my Patreon account? Link your Patreon account via the bot’s Patreon setup command. Once linked, the bot will verify your subscription status and assign appropriate access levels automatically.

What happens if my server grows too large? UniShedr enforces usage restrictions for servers exceeding specific member counts to ensure fair usage and encourage monetization. You may be required to subscribe or upgrade your Patreon tier to continue using all features.

What commands does UniShedr offer? UniShedr includes:

Moderation commands (kick, ban, mute, warn)

Utility commands (server info, user info, help)

Subscription management commands

Use /help to see the full list of available commands.

How do I report bugs or request features? You can report issues or request new features via UniShedr’s GitHub repository or Discord support server. Links are provided in the bot’s help message.

Is my data safe with UniShedr? Yes, UniShedr respects your privacy. User data is only stored as necessary for bot operation and subscription verification. No personal data is shared with third parties without consent.

Can I customize UniShedr for my server? Some customization options will be available, including command prefixes, role-based permissions, and marketplace settings. More customization features are planned for future updates.
